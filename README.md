# 概要 Abstruct

HashMapをラッピングしたシンプルなDB。
A simple DB wrapping a HashMap.

## 利用方法 Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
ys_simple_csv = "0.1"
```

## 実行例 Example

  $ git clone https://github.com/YoshizawaShogo/simple_db.git
  $ cd simple_db
  $ cargo run --example example --release
